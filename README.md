## Design

- **Colors:** BrighterMonday-style — warm orange accent (`#E85D04`), dark hero background (`#1a1f2e`), light content area
- **Typography:** IBM Plex Serif (headings) + IBM Plex Sans (body) for a professional look
- **Hero:** Dark section with strong contrast — name, role, divider line, tagline, CTAs, and stat cards
- **Projects:** Each case study has a content block (left) and a visual/screenshot area (right). Placeholder images are from Unsplash; replace with your own dashboard or analysis screenshots.

## Replacing project images

Each project card has an `<img>` in a `<figure class="project-figure">`. To use your own screenshots:

1. Add your images to a folder, e.g. `images/`.
2. In `index.html`, change each `src` from the Unsplash URL to your file, e.g. `images/market-analysis.png`, `images/retention-dashboard.png`, etc.

## Run

Open `index.html` in a browser. No build step. To host: GitHub Pages, Netlify, or Vercel.
